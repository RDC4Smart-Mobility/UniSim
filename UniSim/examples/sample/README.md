# Example

Install SUMO - Simulation of Urban Mobility before setup Unisim.


## Setup
Set virtualenv

```
python -m venv venv
source venv/bin/activate
```

Install UniSim

```
pip install -e ../../
```

## Run
Run with multiple terminal or following commands.

```
python sample_server.py &
python sample_server2.py &
python sample.py
```
